# RecetasApp

Proyecto para Aprender Angular4

## instalacion
Run `npm install`

## Development server

Run `ng serve --open` servidor pruebas.
Navigate to `http://localhost:4200/`.
Los cambios se refrescan segun vayamos cambiando el codigo.

## Servicio Rest

Run `cd server`
Run `node app.js`

 Navigate to `http://localhost:3000/receta/`
